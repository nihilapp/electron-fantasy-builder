# Fantasy Builder

**세계관 구축 도구** — 창작자가 작품·세계관 설정을 체계적으로 정리하고 기록하는 데스크톱 앱입니다.

---

## 이 프로그램은 무엇인가요?

Fantasy Builder는 **작가, 게임 기획자, 만화가, 시나리오 작가** 등 창작자를 위한 **World Building Tool**입니다.  
소설, 게임, 만화, 시나리오 등에서 쓰는 인물, 종족, 지역, 조직, 사건, 전승 같은 설정을 **프로젝트 단위로 모아서 관리**하고, **특성(Traits)**과 **어빌리티(Abilities)**로 캐릭터·생물의 성격을 일관되게 다룰 수 있게 합니다.

- **데스크톱 앱**: Electron 기반으로 Windows / macOS / Linux에서 실행됩니다.
- **오프라인 우선**: 평소에는 인터넷 없이 사용하며, 데이터는 로컬에 저장됩니다. (원격 DB 전환·온라인 프로젝트는 선택 사항입니다.)

---

## 어떤 것을 다루나요?

### 프로젝트별 세계관 관리

- 작품·세계관 하나를 **프로젝트** 하나로 두고, 그 안에서만 쓰는 설정을 따로 관리합니다.
- 프로젝트마다 **장르, 설명, 커버** 등 메타데이터를 붙일 수 있습니다.

### 전역 풀 + 프로젝트 전용 특성·어빌리티

- **전역 풀**: 여러 작품에서 공통으로 쓸 수 있는 특성·어빌리티를 한곳에서 관리합니다.
- **프로젝트 종속**: 해당 작품에서만 쓰는 고유한 특성·어빌리티를 프로젝트 안에 정의합니다.
- **특성(Traits)**: 캐릭터·생물의 성향, 배경, 관계 등을 키워드처럼 붙여서 정리합니다.
- **어빌리티(Abilities)**: 능력 유형, 대미지 타입, 적용 대상 등 게임/설정용 능력을 체계적으로 다룹니다.

### 설정 엔티티와 관계

- **인물(Characters)**, **생물/종족(Creatures)**, **아이템**, **지역**, **국가**, **조직**, **사건**, **전승** 등의 엔티티를 프로젝트 안에서 생성·관리합니다.
- 엔티티마다 **특성·어빌리티를 매칭**해서 “이 캐릭터는 어떤 성향/능력을 가지는지”를 한눈에 파악할 수 있습니다.
- **엔티티 간 관계**(인물–인물, 인물–조직, 국가–조직 등)를 설정하고, 나중에 탐색·시각화할 수 있도록 합니다.

---

## 누구를 위한 프로그램인가요?

| 대상 | 용도 |
|------|------|
| **작가** | 소설·연재의 인물, 세계관, 조직·국가 관계 정리 |
| **게임 기획자** | 캐릭터 스탯, 종족/직업 특성, 스킬·어빌리티 체계 정리 |
| **만화가·시나리오 작가** | 등장인물, 배경 설정, 사건·전승 기록 |

**핵심 니즈**

- 프로젝트(작품)마다 **독립된 설정 공간**
- **특성·어빌리티**로 설정의 특색을 빠르게 파악
- **엔티티 간 관계** 설정 및 탐색

## 시작하기

- **Node.js** 18 이상, **pnpm** 사용을 권장합니다.

```bash
pnpm install
pnpm dev
```

개발·빌드·API·DB 설정 등 **개발자용 상세 내용**은 [PRD](./PRD/PRD.md) 및 `docs/` 폴더를 참고하세요.
